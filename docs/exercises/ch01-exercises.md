# 第1章 练习

## 层次一：基础

### 1.1 循环不变式证明

用循环不变式证明插入排序的正确性：
- 初始化：循环开始前不变式是否成立？
- 保持：每次迭代是否维持不变式？
- 终止：循环终止时不变式是否给出期望结果？

### 1.2 反例构造

证明贪心硬币找零策略不可行：硬币面额 {1, 5, 10, 25}，贪心策略对哪些面额集合有效？对哪些无效？构造反例。

### 1.3 渐近分析

给出 O(n) 和 O(n²) 之间所有常见增长量级，并排序。证明 Θ(n lg n) 严格介于 Θ(n) 和 Θ(n²) 之间。

### 1.4 建模练习

将以下现实问题建模为组合对象（排列/子集/图/树/递归结构）：
- 旅行商问题
- 子集和问题
- 拓扑排序

## 层次二：LLM 协同

### 2.1 循环不变式验证

让 LLM 实现一个排序算法（如快排的 PARTITION 过程）。用循环不变式检查其正确性：
- 要求 LLM 声明循环不变式
- 检查初始化、保持、终止三步是否成立
- 找出 LLM 代码中的微妙错误（如果有）

### 2.2 六问法分析

用六问法分析一个你不熟悉的问题（如活动选择问题），记录每步推理：
- 哪些步骤 LLM 能独立回答？
- 哪些步骤需要人的判断？
- 为什么建模和问题理解步骤特别依赖人？

## 层次三：Agent 设计

### 3.1 问题分类 Skill

设计一个 Skill：输入问题描述，执行六问法的第3步（查问题目录），输出问题分类和推荐范式。

要求：
- 建立一个问题分类目录（排序、查找、图、DP、贪心、NP完全等）
- 对输入的问题描述进行关键词匹配和语义分析
- 输出最可能的问题类别和推荐的算法范式
