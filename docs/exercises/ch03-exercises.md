**1.1** 三元悖例子：计数排序(O(n)时间+稳定+不原地)、归并排序(O(nlgn)时间+稳定+不原地)、堆排序(O(nlgn)时间+不稳定+原地)。不存在同时满足三者的比较排序。

**2** 分组大小 k=5：T(n) ≤ T(n/5) + T(7n/10) + O(n)。T(n/5) 的中位数保证至少 3n/10 元素被排除。由 Akra-Bazzi 方法得 T(n) = O(n)。k=3 时 T(n) ≥ T(n/3) + T(2n/3) + O(n) = O(n lg n)。
