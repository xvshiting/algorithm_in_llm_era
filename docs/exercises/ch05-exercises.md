# 第5章 练习

## 层次一：基础

### 1.1 回溯框架

用回溯五子程序框架实现：
- 子集生成（n 元素的所有子集）
- 排列生成（n 元素的所有排列）

比较两者的解空间大小和运行时间。

### 1.2 剪枝效果

实现 Sudoku 求解器，比较有剪枝和无剪枝的步骤数差异。最受约束优先剪枝能减少多少搜索量？

### 1.3 A* 手动追踪

给定一个 TSP 小实例（5 个城市），手动追踪 A* 的优先队列状态：
- 选择可容许下界估计
- 比较回溯和 A* 的搜索节点数

### 1.4 模拟退火参数

分析 SA 的温度参数对搜索行为的影响：
- 初始温度 T₀ 太高会怎样？
- 冷却速率太快会怎样？
- 为什么 e^{ΔE/kT} 的接受概率在高温时接近随机、低温时接近贪心？

## 层次二：LLM 协同

### 2.1 搜索策略对比

让 LLM 分别用贪心、回溯、SA 求解同一组合优化问题（如 Clique 或 Set Cover），对比：
- 解的质量
- 运行时间
- 是否保证最优

### 2.2 构造局部最优陷阱

构造一个让 LLM 的贪心策略陷入局部最优的问题实例。要求：
- 贪心解明显劣于全局最优
- 提示 LLM 用回溯或 SA 能否找到更优解

## 层次三：Agent 设计

### 3.1 搜索策略自动选择 Skill

设计一个 Agent：输入问题特征，自动选择搜索策略并调参。

输入特征：
- 解空间大小
- 约束强度
- 是否需要最优解
- 评估函数是否容易计算

策略映射：
- 小空间 + 强约束 + 需最优 → 回溯+剪枝
- 大空间 + 弱约束 + 近似可接受 → SA
- 有好的可容许下界 → A*
- 大空间 + 有结构 → 分支限界
