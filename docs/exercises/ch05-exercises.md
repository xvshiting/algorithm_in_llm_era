# 第5章 练习

## 证明

1. 证明 Sudoku 的剪枝搜索可以从 1,904,832 步降到 48 步——最受约束优先剪枝的效果

## 实现

2. 用回溯五子程序框架实现子集生成和排列生成，比较解空间大小
3. 实现 Sudoku 求解器，比较有剪枝和无剪枝的步骤数差异
4. 给定 TSP 小实例（5城市），手动追踪 A* 的优先队列状态

## 思考

5. 搜索策略与 Agent 推理策略的对应不是外在映射而是同构思想。分析以下对应的结构相似性和差异：
   - 贪心解码 ↔ 局部搜索
   - 思维链 ↔ DFS（线性展开推理路径）
   - 思维树 ↔ 回溯+剪枝（多分支探索）
   - 思维图 ↔ A*（评估每条推理路径的前景）
   
   贪心解码陷入重复意味着什么？思维树需要什么剪枝条件？启发式函数从哪来？

6. 构造一个让贪心搜索陷入局部最优的组合优化问题，分别用回溯和 SA 求解对比
