## 思考

6. DP 记忆化 ↔ KV-cache 的同构关系：
   - DP 存储已计算的子问题解；KV-cache 存储已计算的公共前缀
   - 两者都是空间换时间——什么情况下缓存无效？（快速排序的子问题不重叠，推理中无 prefix reuse）
   - 识别最优子结构需要人的判断——识别 KV-cache 的有效模式也需要人的决策
