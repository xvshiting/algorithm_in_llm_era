# 第7章 练习

## 层次一：基础

### 1.1 交换论证

用交换论证证明活动选择贪心的正确性：
- 设最优解中最早完成的活动不在贪心选择中
- 将最优解中的活动替换为贪心选择 → 不更差

### 1.2 Huffman 编码

给定频率集合 {a:5, b:9, c:12, d:13, e:16, f:45}：
- 构造 Huffman 树
- 计算每个字符的编码
- 计算加权平均编码长度和压缩比

### 1.3 贪心失败实例

给出 0-1 背包问题上贪心策略（按价值密度排序）失败的实例：
- 构造一个贪心解非最优的实例
- 解释为什么分数背包贪心有效但 0-1 背包贪心失败

### 1.4 拟阵

验证图拟阵 M_G = (S, I_G) 满足遗传性和交换性：
- 遗传性：独立集的子集仍独立
- 交换性：|A| < |B| 且 A,B 独立 → ∃x∈B-A 使 A∪{x} 独立

证明 GREEDY(M, w) 返回最大权独立集。

## 层次二：LLM 协同

### 2.1 贪心 vs DP 对比

让 LLM 对活动选择问题分别给出贪心解和 DP 解：
- 运行两个算法，对比结果和时间复杂度
- 无权活动选择：贪心已最优
- 加权活动选择：贪心是否仍最优？为什么？

### 2.2 让 LLM 发现贪心陷阱

让 LLM 在加权活动选择上用贪心策略，观察其失败。然后让 LLM 自己分析为什么贪心不够：
- 贪心选择性质是否成立？
- 最优子结构是否成立？
- 局部最优是否传递到全局最优？

## 层次三：Agent 设计

### 3.1 算法范式选择器 Skill

设计一个算法范式选择器 Skill：输入问题描述，判断适合贪心还是 DP。

决策流程：
1. 问题是否有最优子结构？→ 否：考虑其他范式
2. 是否有贪心选择性质？→ 用交换论证验证
3. 交换论证是否成功？→ 是：贪心；否：DP
4. 子问题是否重叠？→ 是：DP；否：分治

输出：推荐范式 + 理由 + 建议的实现框架。
